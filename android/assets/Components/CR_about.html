The GK Electronik app allows user to identify electronic component by scanning them with their mobile device's camera.

Key Functionality

  • Display recognition query status
  • Request component information meta data based on recognition response
  • Render component information on target
  • Transition display of component information to screen when off target

Targets

Sample Books

Instructions

  • Point camera at component to view info
  • Click on link in description for more info on the component
  • Press 'OK' button to scan another book
